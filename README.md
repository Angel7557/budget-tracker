💰 BudgetFlow – Behavior-Based Smart Budget Tracker

🔗 Live Demo: https://angel7557.github.io/budget-tracker/

🚀 Overview

BudgetFlow is a modern, multi-page budget tracking web application designed to help users not only track expenses but also understand and improve their spending behavior.

Unlike traditional expense trackers, this system integrates insights, challenges, team tracking, and location-based analysis to create a smarter financial experience.

✨ Key Features
💰 Add & manage expenses with categories
📊 Interactive dashboard with visual analytics
🌍 Location-based expense tracking (Map view)
🎯 Savings goals & No-Spend challenge
👥 Team leaderboard for shared expense tracking
🧠 Smart insights for spending behavior analysis
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Storage: Browser LocalStorage
Charts: Chart.js
Map Integration: Leaflet.js
Logic: Rule-based analysis (AI-inspired insights)
💡 How It Works
Users add expenses with details and optional location
Data is stored in localStorage
Dashboard visualizes data using charts
Map shows spending locations
Insights analyze patterns and provide suggestions
🌟 Unique Features
Behavior-based financial analysis
Location-based spending visualization
Gamification (challenges & goals)
Team-based expense comparison
⚠️ Limitations
No real authentication system (currently single-user)
Uses localStorage (no cloud sync)
Map visualization is basic (can be enhanced further)
🚀 Future Scope

🎯 Conclusion

BudgetFlow demonstrates how financial tools can evolve from simple tracking apps into intelligent systems that guide users toward better financial decisions.
